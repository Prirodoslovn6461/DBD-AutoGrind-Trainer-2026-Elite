## 🔍 Troubleshooting common issues

If you encounter problems, check these solutions:

* The trainer does not open: Ensure you have the latest Microsoft Visual C++ Redistributables installed.
* Features do not activate: Make sure you ran the tool as an administrator. Right-click the executable and check the Properties menu to verify the run mode.
* Overlays are not visible: Run Dead by Daylight in Borderless Windowed mode. Exclusive Fullscreen sometimes prevents overlays from rendering on top of the game.
* The application crashes: Restart both the game and the trainer. Ensure no other game-altering software runs at the same time.
